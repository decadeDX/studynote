## 三、推荐学习路径

### 前端学习路径
```
HTML/CSS → JavaScript(ES6+) → TypeScript → React/Vue → 构建工具(Vite/Webpack) 
→ 路由/状态管理 → 前端工程化 → Node.js → 全栈框架(Next.js/Nuxt)
```

### 后端学习路径（Java 方向）
```
Java SE → MySQL → JDBC → Spring Boot → MyBatis → Redis → Spring Cloud 
→ Docker → 消息队列 → 微服务架构 → K8s
```

### 后端学习路径（Go 方向）
```
Go 基础 → MySQL → Gin → GORM → Redis → gRPC → Docker → 微服务架构 → K8s
```

### 后端学习路径（Node.js 方向）
```
JavaScript/TS → Node.js → Express/NestJS → MySQL(MongoDB) → Prisma → 
Redis → Docker → 全栈(Next.js)
```
