## 二、中级软件设计师

### 考试信息
| 项目   | 内容                         |
| ---- | -------------------------- |
| 考试时间 | 每年5月、11月（目标11月）            |
| 上午科目 | 基础知识（75道选择题，150min）        |
| 下午科目 | 案例分析+设计（6道主观大题选做5道，150min） |
| 合格标准 | 两科均≥45分（满分75分/科）           |
|      |                            |

### 考试五大模块
1. **计算机基础**：数据结构、操作系统、计算机网络、计算机组成
2. **数据库技术**：ER图、SQL、范式、事务管理
3. **程序设计**：面向对象设计、设计模式、UML
4. **软件工程**：开发模型、软件测试、项目管理
5. **Web开发**（下午选做题）：HTML/CSS/JS，Java/Python基础

### 1️⃣ 核心资料（必备）

| 资料 | 推荐版本 | 用途 | 预计费用 |
|------|----------|------|----------|
| **官方教材** | 清华大学出版社《软件设计师教程》（第5版） | 考点全覆盖，考试大纲依据 | ¥70~90 |
| **真题卷** | 历年真题（近5年，10套以上） | **最重要！** 题型固定，真题反复刷 | ¥30~50 |
| **核心考点梳理** | 《软件设计师考试32小时通关》（机械工业出版社） | 快速过知识点，比教材薄很多 | ¥40~60 |

### 2️⃣ 辅助资料（按需）

| 资料 | 说明 |
|------|------|
| **王道的考研教材** | 数据结构、操作系统、计网章节如果自学有困难，用王道教材补基础（与软考高度重叠） |
| 软考通APP | 碎片时间刷上午选择题，自动收录错题 |
| 51CTO/希赛网 | 有免费公开课和真题解析视频 |
| B站UP主"软考加油""每天一个软考题" | 零散知识点讲解，通勤听 |

### 3️⃣ 午门专项突破

| 题型 | 得分策略 | 推荐练习方式 |
|------|---------|------------|
| **数据流图（DFD）** | 每年必考，套路固定 | 刷近5年真题，总结画图模板 |
| **数据库设计（ER→关系模式）** | 必考，ER图转SQL | 真题+教材课后习题 |
| **面向对象设计（UML+Java/C++）** | 必考，设计模式 | 重点记工厂、单例、观察者等常见模式 |
| **算法与C语言** | 难度较大，性价比低 | 掌握基础递归、排序即可，不强求满分 |
| **Web开发（选做）** | 适合有前端基础的选做 | HTML/CSS/JS基础，session/cookie概念 |

### 4️⃣ 备考节奏参考
| 阶段 | 资料使用方式 |
|------|-------------|
| 9月基础期 | 通读《32小时通关》+ 教材对应章节，周末做1套分模块练习题 |
| 10月刷题期 | 周末上午下午各1套真题，限时模拟，复盘错题知识点 |
| 11月冲刺期 | 重点刷下午大题模板，背上午选择题高频考点（进制、操作系统、计网协议） |

## 三、中小学教师资格证

### 考试信息
| 项目 | 内容 |
|------|------|
| 笔试时间 | 每年3月、11月（目标次年3月笔试） |
| 面试时间 | 每年5月、次年1月（目标次年5月面试） |
| 适用学段 | 初级中学/高级中学 |
| 笔试科目 | 科目一《综合素质》+ 科目二《教育知识与能力》（中学） |
| 面试形式 | 结构化问答(5min) + 试讲(10min) + 答辩(5min) |

### 1️⃣ 核心资料（必备）

| 资料 | 推荐版本 | 用途 | 预计费用 |
|------|----------|------|----------|
| **教材套装** | 中公教育《综合素质》+《教育知识与能力》（中学版） | 考点系统学习 | ¥60~80 |
| **真题卷** | 中公/粉笔历年真题（近3年，6套） | 熟悉题型和出题风格 | ¥30~40 |
| **高频简答题汇总** | 粉笔/中公的冲刺背诵手册，或直接找PDF | 科目二核心背诵材料 | ¥20~30 |
| **面试指导书** | 中公/粉笔《面试一本通》 | 了解流程、试讲模板 | ¥30~40 |

### 2️⃣ 辅助资料（按需）

| 资料 | 说明 |
|------|------|
| 粉笔教师APP | 刷选择题+简答题背诵+面试视频，**强烈推荐免费版** |
| B站面试试讲视频 | 搜"教资面试试讲示范"，看试讲流程和板书 |
| 科目一写作范文 | B站/公众号整理的大作文模板，考前一周速背 |

### 3️⃣ 各科目备考要点

#### 科目一《综合素质》
| 模块 | 占比 | 备考策略 |
|------|------|---------|
| 职业理念（教育观/学生观/教师观） | ~15% | 背三观模板，材料分析题直接套用 |
| 教育法律法规 | ~10% | 重点法条：教师法、教育法、未成年人保护法 |
| 教师职业道德 | ~15% | **必考大题**，背"三爱两人一终身" |
| 文化素养 | ~12% | 范围太广，随缘复习，不投入大量时间 |
| 基本能力（逻辑+信息+写作） | ~48% | **作文50分是重心**，背议论文结构模板 |

#### 科目二《教育知识与能力》
| 模块 | 占比 | 备考策略 |
|------|------|---------|
| 教育基础知识和原理 | ~20% | 理解为主，选择题送分 |
| 中学课程/教学 | ~20% | 课程类型、教学原则是重点简答题方向 |
| 中学生学习心理 | ~20% | 认知、学习理论常考辨析题 |
| 中学德育 | ~15% | 德育原则+方法，高频简答 |
| 班级管理 | ~10% | 班主任工作、班集体建设，简答/材料题 |

#### 面试
| 环节 | 备考策略 |
|------|---------|
| 结构化问答 | 背4大类模板（自我认知、应急应变、人际关系、综合分析） |
| 试讲 | 固定流程：导入→新授→巩固→小结→作业。练习板书+讲台仪态 |
| 答辩 | 根据试讲内容追问，答"改进方向"比"没毛病"更真诚 |

### 4️⃣ 备考节奏参考
| 阶段 | 资料使用方式 |
|------|-------------|
| 寒假（1月） | 通读教材，做思维导图，了解科目一和科目二的框架 |
| 2月刷题期 | 粉笔APP刷选择题，背诵高频简答100题，周末做真题卷 |
| 3月考前 | 背作文模板+简答押题，过一遍法条 |
| 4月笔试出分→面试 | 看面试视频，练习试讲（录视频自查），背结构化模板 |
| 5月面试 | 保持每天1次模拟试讲，控制10分钟节奏 |

### 5️⃣ 零成本获取渠道
- B站UP主"粉笔教师资格证""中公教育" → 系统课免费看
- 小红书 → 搜"教资简答必背"有整理好的PDF
- 微信公众号"教师资格证考试" → 推送简答和时政考点
