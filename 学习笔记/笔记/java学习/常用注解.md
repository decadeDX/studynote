# 1. Lombok

```java
@Slf4j   // 自动生成 log 字段（SLF4J Logger），直接使用 log.info/warn/error
log.info("这是一条日志");
log.warn("WARN级别");
log.error("ERROR级别");
```

```java
@Data                   // @Getter + @Setter + @ToString + @EqualsAndHashCode + @RequiredArgsConstructor
@NoArgsConstructor      // 生成无参构造方法
@AllArgsConstructor     // 生成全参构造方法
@Builder                // 建造者模式 Builder
@Getter / @Setter       // 单独为字段生成 getter/setter
@ToString               // 生成 toString 方法
@EqualsAndHashCode      // 生成 equals 和 hashCode
@RequiredArgsConstructor // 为 final / @NonNull 字段生成构造方法
@Value                  // 不可变类（所有字段 final，仅有 getter 无 setter）
@With                   // 生成返回新副本的 withXxx 方法
@Accessors(chain = true) // 链式调用 setter（.name("A").age(18)）
```

```java
@NonNull                // 方法参数／字段不能为 null（Lombok 会在生成的方法中做空检查）
```

# 2. Spring 核心 (IOC/DI)

## 声明 Bean

```java
@Component          // 通用组件，标记类为 Spring Bean
@Controller         // 控制器层（Web 层）
@Service            // 业务逻辑层
@Repository         // 数据访问层（DAO）
@Configuration      // 配置类（替代 XML）
@SpringBootApplication // 启动类 = @Configuration + @EnableAutoConfiguration + @ComponentScan
@Primary            // 同类型多 Bean 时优先注入
```

## 注入 Bean

```java
@Autowired          // 按类型注入（Spring 标准）
@Qualifier("beanName") // 配合 @Autowired 指定 Bean 名称
@Resource(name = "beanName") // 按名称注入（Jakarta 标准）
@Inject             // 按类型注入（Jakarta 标准，需 javax.inject）
```

## 配置与属性绑定

```java
@Bean               // 方法返回对象注册为 Spring Bean
@Scope("prototype") // 指定作用域（singleton / prototype / request / session）
@Lazy               // 延迟初始化
@Value("${app.name}") // 注入配置文件（application.yml）中的值
@ConfigurationProperties(prefix = "app.datasource") // 绑定配置前缀到对象字段
@PropertySource("classpath:db.properties") // 指定加载的配置文件
@Import(OtherConfig.class) // 导入其他配置类
```

# 3. Spring MVC / Web

## 请求映射

```java
@RequestMapping("/api/v1")    // 类/方法级别路径映射（可指定 method）
@GetMapping("/users")         // GET 请求
@PostMapping("/users")        // POST 请求
@PutMapping("/users/{id}")    // PUT 请求
@DeleteMapping("/users/{id}") // DELETE 请求
@PatchMapping("/users/{id}")  // PATCH 请求（部分更新）
```

## 参数提取

```java
@RequestHeader("Authorization")        // 提取请求头参数
@RequestParam("page")                   // 提取 URL 查询参数
@RequestBody                            // 提取请求体（JSON），绑定到自定义对象
@PathVariable("id")                     // 提取路径参数 /users/{id}
@CookieValue("sessionId")               // 提取 Cookie 值
@RequestPart("file")                    // 提取 multipart 文件上传
@MatrixVariable                         // 提取矩阵变量
```

## 响应与全局处理

```java
@RestController         // = @Controller + @ResponseBody，类中所有方法返回 JSON
@ResponseBody           // 方法返回值直接作为 HTTP 响应体
@CrossOrigin            // 允许跨域（可指定 origins / methods）
@ExceptionHandler(XxxException.class) // 处理当前 Controller 中的特定异常
@ControllerAdvice       // 全局异常处理 / 数据绑定 / 数据预处理
@RestControllerAdvice   // = @ControllerAdvice + @ResponseBody
@InitBinder             // 自定义参数绑定（如日期格式转换）
@ModelAttribute         // 在模型中添加属性（用于视图渲染或参数预处理）
```

```java
// @RequestMapping 示例
@RequestMapping("/api/v1")
@RestController
public class UserController {

    @GetMapping("/hello")
    public String hello() {
        return "Hello";
    }
}
```

# 4. Spring 事务

```java
@Transactional                              // 声明事务（类或方法级别）
@Transactional(readOnly = true)             // 只读事务，优化查询性能
@Transactional(rollbackFor = Exception.class)      // 指定回滚异常类型（默认只回滚 RuntimeException）
@Transactional(noRollbackFor = ...)                 // 指定不回滚的异常
@Transactional(timeout = 5)                 // 超时时间（秒）
@Transactional(propagation = Propagation.REQUIRES_NEW) // 事务传播行为
@EnableTransactionManagement                // 启用注解式事务管理（通常放在配置类）
```

# 5. Spring 定时与异步

```java
@Scheduled(cron = "0 0 9 * * ?")         // 定时任务（cron 表达式）
@Scheduled(fixedRate = 5000)              // 固定频率执行（毫秒，不考虑任务耗时）
@Scheduled(fixedDelay = 5000)             // 上次执行完后固定延迟再执行
@Scheduled(initialDelay = 10000)          // 延迟首次执行时间（毫秒）
@EnableScheduling                         // 启用定时任务
```

```java
@Async                                    // 异步执行该方法（调用方立即返回）
@EnableAsync                              // 启用异步支持
```

# 6. Spring 缓存

```java
@Cacheable(value = "users", key = "#id")         // 缓存结果（下次同参数直接返回缓存）
@CachePut(value = "users", key = "#user.id")     // 更新缓存（方法总会执行）
@CacheEvict(value = "users", key = "#id")        // 清除指定缓存
@CacheEvict(value = "users", allEntries = true)  // 清除该缓存区所有条目
@Caching(put = { ... }, evict = { ... })         // 组合多个缓存操作
@CacheConfig(cacheNames = "users")               // 类级别缓存公共配置
@EnableCaching                            // 启用缓存
```

# 7. Spring 验证 (Validation)

## 启用验证

```java
@Valid              // 在 Controller 参数上启用 Bean Validation（Jakarta 标准）
@Validated          // Spring 增强版，支持分组校验
```

## 常见校验注解

```java
@NotNull            // 字段不能为 null
@NotEmpty           // 字符串/集合不能为 null 且长度/大小 > 0
@NotBlank           // 字符串不能为 null 且去除空白后长度 > 0
@Size(min = 2, max = 30)   // 限制字符串/集合大小范围
@Min(1)             // 数字最小值
@Max(100)           // 数字最大值
@Email              // 邮箱格式校验
@Pattern(regexp = "^1[3-9]\\d{9}$") // 正则表达式匹配
@Positive           // 必须为正数
@PositiveOrZero     // 必须为 >= 0
@Negative           // 必须为负数
@NegativeOrZero     // 必须为 <= 0
@Past               // 日期必须为过去时间
@Future             // 日期必须为将来时间
@AssertTrue         // 必须为 true（用于业务逻辑校验）
@AssertFalse        // 必须为 false
```

# 8. Spring AOP

```java
@Aspect                             // 声明切面类
@EnableAspectJAutoProxy             // 启用 AOP 代理（配置类上）
```

```java
// 通知类型
@Before("execution(* service.*.*(..))")          // 方法执行前
@After("execution(* service.*.*(..))")           // 方法执行后（无论成功/异常）
@AfterReturning("execution(* service.*.*(..))")  // 方法成功返回后
@AfterThrowing("execution(* service.*.*(..))")   // 方法抛出异常后
@Around("execution(* service.*.*(..))")          // 环绕通知（最强大，可控制方法执行）
```

```java
// 切点复用
@Pointcut("execution(* com.example.service.*.*(..))")
public void serviceLayer() {}

@Before("serviceLayer()")
public void beforeService() {
    // ...
}
```

# 9. Jackson (JSON 处理)

```java
@JsonProperty("user_name")                     // 指定 JSON 字段名称
@JsonIgnore                                    // 忽略该字段，不参与序列化/反序列化
@JsonIgnoreProperties(ignoreUnknown = true)    // 类级别，忽略未知字段
@JsonFormat(pattern = "yyyy-MM-dd HH:mm:ss")   // 日期格式化
@JsonInclude(JsonInclude.Include.NON_NULL)     // 只输出非 null 字段
@JsonAlias({"username", "userName"})           // 反序列化时接受多个别名
@JsonNaming(PropertyNamingStrategies.SnakeCaseStrategy.class) // 统一命名策略（驼峰→蛇形）
@JsonView(Views.Public.class)                  // 视图控制，不同场景返回不同字段
@JsonUnwrapped                                 // 展开嵌套对象字段到父层
@JsonAnySetter / @JsonAnyGetter                // 处理动态属性
@JsonSerialize(using = CustomSerializer.class) // 自定义序列化器
@JsonDeserialize(using = CustomDeserializer.class) // 自定义反序列化器
```

```java
// 使用示例
ObjectMapper objectMapper = new ObjectMapper();
String json = objectMapper.writeValueAsString(person); // 对象 → JSON
Person p = objectMapper.readValue(json, Person.class); // JSON → 对象
```

# 10. JPA (持久化)

## 实体定义

```java
@Entity(name = "User")                       // 声明为 JPA 实体
@Table(name = "t_user")                      // 指定数据库表名
@Id                                          // 主键
@GeneratedValue(strategy = GenerationType.IDENTITY) // 主键生成策略（自增/序列/UUID）
@Column(name = "user_name", nullable = false, length = 50) // 字段映射
@Transient                                   // 忽略该字段，不持久化到数据库
@Enumerated(EnumType.STRING)                 // 枚举映射为字符串
@Temporal(TemporalType.TIMESTAMP)            // 日期类型映射（DATE / TIME / TIMESTAMP）
@Lob                                         // 大字段（CLOB 文本 / BLOB 二进制）
@CreationTimestamp / @UpdateTimestamp        // 自动时间戳（Hibernate 特有）
```

## 关联关系

```java
@OneToOne(mappedBy = "user")                 // 一对一
@OneToMany(mappedBy = "user")                // 一对多（一方）
@ManyToOne                                   // 多对一（多方）
@ManyToMany                                  // 多对多
@JoinColumn(name = "dept_id")                // 指定外键列
@JoinTable(name = "user_role",               // 多对多中间表
    joinColumns = @JoinColumn(name = "user_id"),
    inverseJoinColumns = @JoinColumn(name = "role_id"))
```

## 查询与审计

```java
@NamedQuery(name = "User.findByName", query = "SELECT u FROM User u WHERE u.name = :name")
@OrderBy("createTime ASC")                   // 关联集合排序
@EntityListeners(AuditingEntityListener.class) // 实体监听器（审计）
@CreatedDate / @LastModifiedDate            // 创建时间 / 修改时间
@CreatedBy / @LastModifiedBy                // 创建人 / 修改人
@EnableJpaAuditing                          // 启用 JPA 审计
```

# 11. 测试 (Testing)

## JUnit 5

```java
@Test                              // 标记测试方法
@ParameterizedTest                 // 参数化测试
@RepeatedTest(5)                   // 重复执行 N 次
@BeforeEach                        // 每个 @Test 前执行
@AfterEach                         // 每个 @Test 后执行
@BeforeAll                         // 所有测试前执行一次（static）
@AfterAll                          // 所有测试后执行一次（static）
@DisplayName("用户登录成功场景测试")    // 自定义测试名称
@Disabled                          // 禁用测试
@Timeout(5)                        // 超时限制（秒）
@Order(1)                          // 测试方法执行顺序
```

## Spring 测试

```java
@SpringBootTest                    // 启动完整 Spring 上下文
@DataJpaTest                       // 只加载 JPA 组件，测试 Repository 层
@WebMvcTest(UserController.class)  // 只加载 Web 层，测试 Controller
@RestClientTest                    // 测试 REST 客户端
@MockBean                          // 模拟 Bean（替换上下文中的真实 Bean）
@SpyBean                           // 部分模拟 Bean
@AutoConfigureMockMvc              // 自动配置 MockMvc
@AutoConfigureTestDatabase(replace = Replace.NONE) // 使用真实数据库而非嵌入式
@TestConfiguration                 // 测试专用的配置类
@ActiveProfiles("test")            // 激活指定环境配置
@Sql(scripts = "test-data.sql")    // 测试前执行 SQL 脚本
@SqlGroup({ ... })                 // 组合多个 @Sql
```

```java
// MockMvc 示例
@Autowired
private MockMvc mockMvc;

@Test
void testGetUser() throws Exception {
    mockMvc.perform(get("/api/users/1"))
           .andExpect(status().isOk())
           .andExpect(jsonPath("$.name").value("小明"));
}
```

## Mockito

```java
@Mock                              // 创建 Mock 对象
@Spy                               // 创建 Spy 对象（部分模拟，保留真实行为）
@InjectMocks                       // 将 @Mock 对象注入到被测类
@Captor                            // 捕获方法参数（用于断言）
```

# 12. Java 标准注解

```java
@Override               // 重写父类/接口方法，编译器检查签名一致性
@Deprecated             // 标记已弃用，调用处收到编译警告
@SuppressWarnings("unchecked")  // 抑制指定编译警告
@FunctionalInterface    // 声明函数式接口（只能有一个抽象方法）
@SafeVarargs            // 声明可变参数操作是安全的（抑制堆污染警告）
@Generated("value")     // 标记为自动生成的代码
```
